# ai-regulation-comparator

This project compares selected provisions of the EU Artificial Intelligence Act (EU AI Act)
and Turkish data protection law (KVKK) using NLP-based semantic similarity analysis and
basic concept mapping.

It is intended as a lightweight tool to support comparative legal and regulatory analysis
in the field of artificial intelligence.

Bu proje, Avrupa Birliği Yapay Zeka Yasası (EU AI Act) ile Türk veri koruma mevzuatı (KVKK)
arasında NLP tabanlı anlamsal benzerlik analizi ve temel kavram eşleştirmesi yoluyla
karşılaştırma yapan hafif bir Python aracıdır.

Yapay zeka alanındaki hukuki ve düzenleyici karşılaştırmalı analizleri desteklemek amacıyla
geliştirilmiştir.
